# Crowdsec with Traefik Bouncer

Let's protect your homelab with Crowdsec and Traefik Bouncer

## Setup

Let's setup directories in your homefolder like so:
- ~/crowdsec
    - compose.yml
    - config
        - acquis.yaml

Basically:
```
mkdir crowdsec
```
```
cd crowdsec
```
```
touch compose.yml
```
```
mkdir config
```
```
touch acquis.yaml
```
